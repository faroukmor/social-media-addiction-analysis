# Social Media Addiction Classification

A machine learning project that classifies social media addiction levels (High, Medium, Low) based on behavioral and psychological metrics using a Logistic Regression Pipeline.

## 📊 Model Performance
- **Accuracy:** 97%
- **Macro Average F1-Score:** 0.97
- Key predictive features include `dopamine_dependency_score` and `attention_span_score`.
